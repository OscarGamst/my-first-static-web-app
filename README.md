## 🚀 Getting Started
### 1️⃣ Clone the Repository
- git clone https://github.com/OscarGamst/APP2000-Group2.git  
- cd APP2000-Group2  
##



### 2️⃣ Change directory
- cd APP2000-Group2   
##



### 3️⃣ Install Dependencies
- npm install  
##



### 4️⃣ Run locally
- npm start
